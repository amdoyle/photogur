
*Photogur*

Description
* Basic rails clone of Imgur
* Allows users to line to images and provide the photographers name and a title

Versions
* Ruby version 2.3.0
* Rails version 4.2.5

Gems
* No additional gems, beyond the default gems included in rails

Set-up
* Instal gems
* Migrate models


Additional
* No additional set-up required
